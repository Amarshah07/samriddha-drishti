What is Samriddha Drishti?

Samriddha Drishti is an AI-powered solution designed to assist visually impaired individuals through smart glasses. The glasses use advanced technologies like object detection, face recognition, and scene description to help users navigate and interact with their surroundings more effectively.

Features:

  • Object Detection
  • Face Recognition
  • Text Reading (OCR)
  • Voice Assistance
  • Navigation Support
